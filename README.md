# DOIS Studio

A multi-tenant SaaS application for field inspectors. It is a highly customized CRM, ERP, and routing dashboard.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
